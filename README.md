# smartcam
SMARTCAM AI is a real-time computer vision platform built with Python, Flask, OpenCV, MediaPipe, and DeepFace. It transforms a standard webcam into an intelligent vision system capable of live facial tracking. Designed and Developed by Divyansh Tiwari.  © 2026 Divyansh Tiwari. All Rights Reserved.

SMARTCAM AI is a real-time computer vision platform built with Python, Flask, OpenCV, MediaPipe, and DeepFace. It transforms a standard webcam into an intelligent vision system capable of live facial tracking, face mesh detection, emotion analysis, eye direction estimation, and multi-face monitoring through a modern AI dashboard.

The project features a futuristic glassmorphism-inspired interface with animated transitions, live system status, responsive design for desktop and mobile devices, and a clean user experience inspired by modern AI systems. SMARTCAM AI is designed as both a learning project and a foundation for advanced AI-powered surveillance, monitoring, and vision applications.

Current Features:
• Real-time webcam streaming
• Multi-face detection and tracking
• MediaPipe Face Mesh visualization
• DeepFace emotion recognition
• Eye direction estimation
• AI system dashboard
• Responsive web interface
• Animated startup sequence
• Live FPS and system monitoring

Planned Features:
• Face recognition with known identities
• Object detection (YOLO)
• Voice assistant integration
• AI event logging
• Screenshot and recording tools
• Smart notifications
• Mobile PWA support
• Cloud deployment

This project demonstrates the integration of computer vision, machine learning, and modern web technologies into a single interactive application.
